# PROMPT FOR GEMINI — Infacta favicon / logo mark concepts

Ти — бренд-дизайн reviewer. Потрібно запропонувати 5–7 концепцій favicon / small logo mark для Infacta.

Контекст Infacta:
- сервіс попереднього структурування, пояснення й аналізу документів;
- не юридична фірма і не заміна юриста;
- ключові асоціації: документ, довіра, структура, перевірка, ясність, доказовість, зони уваги;
- стиль сайту: archival paper / document trust, теплий паперовий фон, стриманий navy/gold, мінімалізм;
- бренд: inFACTa, акцент на facts / фактах / структурі інформації;
- favicon має бути сучасним, чітким і читабельним у 16×16, 32×32, 48×48.

Завдання:
1. Запропонуй 5–7 різних напрямів favicon/logo mark.
2. Для кожного напряму дай:
   - коротку назву концепції;
   - що символізує;
   - чому підходить Infacta;
   - які форми/кольори використати;
   - ризики: чому може не спрацювати в малому розмірі.
3. Не використовуй ваги правосуддя як головний символ: Infacta не юридична консультація.
4. Уникай мутних дрібних деталей і довгого тексту в іконці.
5. Окремо вибери 2 найсильніші концепції для тестового favicon.

Відповідь дай українською.
